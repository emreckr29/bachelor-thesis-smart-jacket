# Physiotherapy App Frontend

This directory contains the **Angular-based frontend application**
of the Smart Jacket physiotherapy system.

The frontend is responsible for user interaction and visualization
of exercise feedback generated by the machine learning models.

## Features

- User interface for physiotherapy exercises
- Communication with backend and ML API
- Visualization of movement correctness and feedback
- Clean separation of UI logic and data processing

## Technologies Used

- Angular
- TypeScript
- HTML / SCSS

## Architecture

The frontend:
1. Sends sensor or processed data to the backend / ML API
2. Receives prediction results
3. Displays feedback to the user in real time

## Notes

- `node_modules` is intentionally excluded via `.gitignore`
- The project can be built and run using standard Angular CLI commands
- This folder represents the **client-side** of the overall system
